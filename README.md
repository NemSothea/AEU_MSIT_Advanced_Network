# 🚀 CCNA Mastery: 10 Real-World Packet Tracer Labs & Lesson Advanced Network

**From Zero to Network Engineer: A Comprehensive Hands-On Roadmap**

Welcome to the ultimate CCNA practical training suite. This repository contains 10 meticulously designed Cisco Packet Tracer labs covering the core pillars of the CCNA 200-301 exam. Don't just read the OCG—build the network.

---

## 🛠️ Included Labs & Technologies

This bundle provides step-by-step configuration and troubleshooting scenarios for:

* **VLANs & Trunking:** Logical segmentation and 802.1Q tagging.
* **Inter-VLAN Routing:** Router-on-a-Stick and Layer 3 Switch configurations.
* **STP (Spanning Tree Protocol):** Preventing loops and understanding Root Bridge election.
* **EtherChannel:** Increasing bandwidth via Link Aggregation.
* **OSPFv2:** Single-area dynamic routing, neighbor adjacencies, and cost tuning.
* **HSRP (First Hop Redundancy):** Gateway redundancy for high availability.
* **ACLs (Standard & Extended):** Traffic filtering and security.
* **NAT (Static & PAT):** Mapping private IP space to the public internet.
* **DHCP & Port Security:** Automated addressing and Layer 2 hardware lockdown.
* **The Enterprise Lab:** A capstone project integrating all the above into one cohesive topology.
